考古学初阶之走向21世纪的中国考古学

考古学发展阶段

中国考古学与世界考古学的发展阶段大致相同，

  1. 以研究古代遗物为主的古器物学；
  2. 以田野考古为基础的近代考古学；
  3. 以全面复原古代人类社会历史为目标的现代考古学。

每一个阶段的理论、方法在许多方面是相似或相通的，但中国考古学有自己的特点。

中国古器物学

中国古器物学，

  1. 出现很早；
  2. 在北宋初具规模、形成金石学；
  3. 研究对象以有铭文的商周青铜器、秦汉以来的石刻为主；
  4. 考证、补充了历史记载；
  5. 跟历史学的关系，比欧洲16世纪兴起的古器物学还要密切一些。

中国近代考古学

中国近代考古学，

  1. 从1921年河南渑池县仰韶村遗址等处的发掘才开始的，比欧洲晚了半个世纪以上；
  2. 与欧洲一样，最初也是由地质学家开展起来的；
  3. 起步较晚，西方考古学已然成熟，没有形成独具特色的理论、方法；
  4. 自己的特点：继承发展了传统金石学和古文字学的成果，表现在对甲骨卜辞、铜器铭文的研究方面。

北京周口店旧石器时代遗址，

  1. 1927年开始发掘；
  2. 发现了十分丰富的北京猿人化石、一大批石器和动物化石，还有长期用火的痕迹；
  3. 中外学者共同工作，涉及地质学、古生物学、古人类学和考古学等许多学科；
  4. 初步奠定了中国史前考古学的基础。

河南安阳北部殷墟遗址，

  1. 1928年开始，前后持续十年之久；
  2. 其规模之大、收获之丰，世界罕见；
  3. 发现了一座商代晚期的都城和一个完整的王陵区，获得了大量实物资料和甲骨刻辞等文字资料；
  4. 把中国古代历史的研究推进到一个崭新的阶段。

调查发掘了许多新石器时代遗址，

  1. 确立了仰韶文化、龙山文化、殷商文化的相对年代关系；
  2. 为中原地区的考古年代学奠定了初步的基础；
  3. 为殷商文化的来源找到了重要的线索；
  4. 成长起来一代考古学家；
  5. 形成了一套行之有效的田野考古方法和管理制度。

中国现代考古学

考古新时期

1950年之前，

  1. 虽然也研究古代历史，但受方法、技术限制，主要是文化史研究；
  2. 包括，考古学文化的特征、起源、分期、发展阶段、相互关系等；
  3. 描述多于解释。

1950年或1960年，进入考古新时期，其特点，

  1. 更加明确以全面复原古代人类社会历史为目标；
  2. 多学科的交叉和现代自然科学技术的广泛应用。

这个时期的考古学称为现代考古学。

考古新思想

20世纪30年代，苏联一批青年学者要求用新的观点和方法全面地研究古代人类社会历史，

  1. 批评传统考古学研究是单纯的器物观；
  2. 批评类型学方法是用生物学观点解释历史；
  3. 要求用辩证唯物主义发展论取代当时流行的传播论。

评价，

  1. 符合考古学的目标和发展方向；
  2. 他们对考古学了解太少，批评过头了。

50年代，清算错误思想、形成具有自己特色的苏联考古学派，

  1. 主张考古学是历史学科的一个部门；
  2. 主张以马克思主义哲学为指导，通过研究实物遗存以恢复人类历史的过去；
  3. 强调与民族学资料结合、利用自然科学技术。

1958年，考古学研究方向大辩论，

  1. 背景，
    1. 50年代；
    2. 经济建设全面开展，促使田野考古工作很快发展起来；
    3. 积累了很多资料而研究工作跟不上去；
    4. 如何用新的观点来进行研究成为待解决的问题。
  2. 学生们，
    1. 不满足教学中“见物不见人”的倾向；
    2. 反对单纯器物观；
    3. 反对把考古学当作一门边缘学科；
    4. 认为应尽可能通过实物遗存恢复人类历史的本来面目。
  3. 大家认识到，考古学，
    1. 是一门历史学科；
    2. 任务是要全面复原自有人类以来的古代社会历史；
    3. 研究手段应尽量科学化、现代化。

在西方，

  1. 英国考古学家戈登•柴尔德很早提出，考古学应把研究人和人类社会作为主要目标；
  2. 美国的瓦特•泰勒等批评传统考古学不能很好地复原人类社会历史。
  3. 60年代初，芝加哥大学一批青年人提出，要建立新考古学，
    1. 目标是复原整个人类社会的历史；
    2. 广泛利用系统论、生态学、环境考古等许多相关学科的方法。

考古新发展

第二次全国性文物普查，

  1. 1981年开始；
  2. 对已知考古遗址进行了更详细的调查；
  3. 新发现了数以万计的各个时期的遗址；
  4. 在此基础上，出版了多卷本《中国文物地图集》。、

田野考古工作，

  1. 要注意配合工程建设，例如，三峡水库、小浪底水库等；
  2. 要根据学科发展的需要选择重点，必要时还可以做一些主动性发掘。

史前考古，

  1. 新发现的旧石器遗址已遍布全国各省；
  2. 对旧石器文化特点和发展谱系形成了新认识；
  3. 全国新石器文化发展谱系已日渐明确；
  4. 初步探索了各时期的社会性质和社会组织结构，对文明起源提出了新看法。

夏商周考古，

  1. 夏商周文化同周围青铜文化的关系，将是今后研究的重要课题；
  2. 1983年，发现偃师尸乡沟商城遗址（断句为：偃师/尸乡沟/商城遗址），对夏商文化分际、商代都城定位具有十分重要的意义；
  3. 陕西周原西周甲骨文、各地大量铜器铭文的发现，推进了古文字的研究。

春秋战国考古，

  1. 春秋战国是中国历史的重要转型期；
  2. 我国至少在春秋晚期便已进入铁器时代，个别铁器的制造可追溯到西周晚期；
  3. 铁器促进农业、手工业发展，使大规模修建城池、水利工程成为可能；
  4. 勘察、发掘了列国都城，对我国早期都城特点已有相当认识。

秦汉考古，

  1. 我国历史进入中央集群阶段；
  2. 是历史考古学的主要内容；
  3. 历代都城的演变，
    1. 是历史发展的一个缩影；
    2. 影响了东南亚一些国家的都城制度；
    3. 城市布局可作为探索城市道路规划的线索；
    4. 近些年兴起了城市考古这一分支学科。
  4. 历代陵墓制度和丧葬习俗，
    1. 勘探和发掘各类墓葬数以万计；
    2. 对于了解历代陵墓制度、丧葬习俗，进而研究各时期社会制度、社会生活提供了宝贵的资料；
    3. 举世知名的考古工作，
      1. 秦始皇陵兵马俑坑和铜车马坑，
      2. 长沙马王堆汉墓，
      3. 广州南越文王墓，
      4. 河南永城汉梁王墓
    4. 丰富了我国的物质文化史、美术史、科技史、中外关系史等。

科技考古，

  1. 对古代科学技术的研究，例如，
    1. 冶金，
    2. 陶瓷，
    3. 纺织，
    4. 造纸，
    5. 天文历法，
    6. 农业科学。
  2. 现代科学技术在考古研究上的应用，例如，
    1. 测定年代的技术，
    2. 探测技术，
    3. 成分分析，
    4. 制造工艺研究，
    5. 保存科学研究，
    6. 动植物遗存研究，
    7. 孢粉分析和植硅石分析，
    8. 环境考古
  3. 筹备成立中国科技考古学会

理论上的活跃，

  1. 国外考古学理论五花八门，国内一些有影响的学者的看法也很不一致；
  2. 理论活跃是件好事，可以让人们有个比较；
  3. 张光直主张，
    1. 理论多元化，
    2. 方法系统化，
    3. 技术国际化。
  4. 判断好坏的标准，
    1. 是否能更有效地处理古代人类社会遗留下来的实物资料，以便从中提取尽可能多的科学的信息；
    2. 是否是研究古代人类社会历史及其发展规律的有效途径或手段。

如何走向21世纪

  1. 改变传统观念。
    1. 传统观念1：把考古学看作是历史学科的一个部门。
    2. 改变传统观念1：考古学就是考古学，
      1. 考古学研究的资料、所采用的理论和方法，都和一般意义上的历史学有很大不同；
      2. 美国人干脆把考古学看作是人类学的一部分；
      3. 科学的发展总是在不断分化和不断相互渗透中实现的。
    3. 传统观念2：考古学同数学、物理学不同，不是普适的，而是一种民族性很强的学科，因此，中国人学中国考古学就够了，学外国考古学没什么用；同样，外国人用外国考古学的理论方法，也很难研究中国考古学。
    4. 改变传统观念2：中国要走向世界，世界也要了解中国，
      1. 中国考古学中，有一些关系到全人类历史发展的重大课题，例如，
        1. 人类的起源和早期发展，
        2. 农业的起源及其对人类社会的影响，
        3. 中国文明的起源及其对世界古代文明的贡献。
      2. 中国也是世界的一部分，需要站在世界历史的高度来研究，例如，
        1. 中国文化有什么特点、在世界古代文明中占有什么样的地位，
        2. 中国历史怎样影响了世界历史、又受到世界历史的哪些影响。
  2. 在全国性考古学规划中把外国考古学作为重点项目纳入，加强三个方面的研究，
    1. 对与中国相邻的、历史上曾经同中国发生过密切关系的国家和地区的考古学研究，例如，
      1. 朝鲜半岛
      2. 日本
      3. 西伯利亚
      4. 东南亚、南亚、中亚国家
    2. 对世界古老文明和中国古代文明的比较研究，以便同中国古代文明进行比较，例如，
      1. 美索不达米亚文明，
      2. 尼罗河文明，
      3. 印度河文明，
      4. 中美洲文明。
    3. 对一些考古学比较发达、理论研究比较活跃的国家，要派人学习、参加他们的研究工作，以便提高考古的现代化水平。
